- 概述：

`Floyd-Warshall` 算法（英语：Floyd-Warshall algorithm），中文亦称弗洛伊德算法或佛洛依德算法，是解决任意两点间的最短路径的一种算法，可以正确处理 *有向图* 或 *负权（但不可存在负权回路）* 的 **最短路径问题**，同时也被用于计算有向图的传递闭包

- 复杂度分析：

Floyd-Warshall算法的时间复杂度为 O (N^3), 空间复杂度为 O (N^2) ;
